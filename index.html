One-time setup (admin only)

This app uses JSONBin.io as a free shared database so everyone's picks sync in real time. You only need to do this once.

Step 1: Go to jsonbin.io → Sign up free → Click "+ New Bin" → paste {} as content → click Create Bin.

Step 2: Copy the BIN ID from the URL (looks like 64abc123ef456) and your API Key from the Access Keys page. Paste them below and click Save.


About data & privacy
All picks are stored in your own private JSONBin — nobody else can access it without your API key. The key is saved in each person's browser localStorage and never sent anywhere except JSONBin.
